Looking to learn the basics of HyperText Markup Language (HTML) & Cascading Style Sheets (CSS) so you can finally develop your own website? You’ve come to the right place!

Before we discuss HTML & CSS you are going to have to download a code editor. I like to use Visual Studio Code but there are many you can choose from, such as Sublime, Atom, or Notepad++.

HTML Basics

HTML’s function is to create the content that will be displayed on the page, not to make it look good, that’s what CSS does. In your selected code editor create a .html file. It’s recommended to name that file “index.html” as it will be the page webservers default to when hosting your website.

Basic .html File Setup

For now, you should only be worried about whats inside the <body> </body> tag, which is where you put all your code that displays on the webpage.

HTML is made up of HTML Elements, these elements can be paragraphs, headers, images, links, and more. HTML Elements are made up of:

  • Opening tag, which defines what type of element you are creating.
  • Content area, where you put any content you want to be displayed.
  • Closing tag, which closes the content section and ends the element.
  • Attributes are used to define content for elements which don't need a content. As an example, image elements have no need for a content area or closing tag as all the information is defined in the opening tag (the attribute section).

Although there are some exceptions, such as the image element, which doesn’t have a content area or a closing tag because it uses an attribute to define the content.

Notice how some elements appear on the same line where others don't? That’s because certain elements are block elements, which appear on new line, and others are inline elements, which appear on the same line.

Some block elements include:

  • Paragraphs
  • Headers
  • List Items

Some inline elements include:

  • Links
  • Images
  • Buttons

CSS Basics

The type of CSS we will be covering is inline CSS. Inline CSS is CSS that is put inside the HTML Elements attribute section. It’s divided into 2 sections:

  • Property, which defines what part of the element is being changed.
  • Value, which defines what said property is being change to.
